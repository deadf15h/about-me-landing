## 🛠 Stack

- **React**
- **SASS**

## 🚀 Quick start

1. Clone:

```bash
git clone https://github.com/deadf15h/about-me-landing.git
```

2. Install dependencies:

```bash
npm i
```

3. Run project:

```bash
npm start
```
